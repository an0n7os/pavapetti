# Pavapetti Web App

This folder contains the complete source code for the Pavapetti Heritage Boutique website.

## Structure
- `/website`: The main React/Vite application.
- `/mock-api-server`: The backend server that provides product data.
- `/lib/api-client-react`: The shared API client library.

## How to Run

### 1. Start the Mock API Server
Open a terminal, navigate to the `mock-api-server` folder, and run:
```bash
npm install
node index.mjs
```
The server will start on `http://localhost:8081`.

### 2. Start the Website
Open another terminal, navigate to the `website` folder, and run:
```bash
npm install
npm run dev
```
The website will be available at `http://localhost:3001`.

### 3. Enjoy!
Visit `http://localhost:3001` in your browser to see the heritage boutique in action.
